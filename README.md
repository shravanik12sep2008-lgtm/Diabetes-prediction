Diabetes Prediction Using Machine Learning

## About Project

This project predicts whether a person has diabetes or not using Machine Learning. The model is trained on medical data such as glucose level, blood pressure, BMI, age, and other health details.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm

- Logistic Regression

- Data Preprocessing
- Data Visualization
- Model Training
- Diabetes Prediction
- Accuracy Evaluation
